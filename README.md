# RulingJs

Just a simple set of useful functions to check validation rules

[![npm](https://img.shields.io/npm/v/rulingjs.svg)](https://www.npmjs.com/package/storybook-class-stories)

## Install

```bash
npm i -S rulingjs
```

## Rules

- `email`
- `isCapital`
- `isCapitalOrNumber`
- `isDefined`
- `maxLength`
- `minLength`
- `NotContain`
- `notEmpty`
- `required`
- `strictLength`
